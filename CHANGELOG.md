# Changelog

Все значимые изменения в проекте BeastStaff будут документироваться в этом файле.

## [2.0.0] - 29.11.2025

### Добавлено

#### Интеграция с LiteBans
- Полная интеграция с системой наказаний LiteBans
- Получение статистики наказаний (баны, муты, кики, предупреждения)
- Поддержка MySQL и SQLite баз данных для LiteBans
- Автоматический поиск SQLite БД LiteBans
- Кеширование статистики для оптимизации производительности
- Telegram команды для просмотра статистики наказаний:
  - `/bs_ban_stats` - статистика наказаний за период (сегодня/неделя/месяц)
  - `/bs_recent_bans` - последние наказания с детальной информацией

#### Telegram интеграция
- Полноценный Telegram бот с long polling для обработки команд
- Команды в Telegram:
  - `/bs_status` - просмотр текущего статуса работы
  - `/bs_set_status <статус>` - изменение статуса работы через Telegram
- Chat ID теперь опциональный - бот работает для личных сообщений без указания chat-id
- Улучшенная обработка ошибок с понятными сообщениями
- Поддержка отправки уведомлений в каналы/группы (опционально)

#### Улучшения системы сессий
- Исправлена логика создания сессий - предотвращение дубликатов
- Сессии создаются только при изменении статуса
- Улучшенное сохранение сессий в YAML файлы

#### Оптимизация размера плагина
- Исключение ненужных файлов из зависимостей (лицензии, документация)
- Фильтрация META-INF файлов
- Уменьшение размера JAR файла на ~2-5 MB

### Изменено

#### Конфигурация
- Упрощена структура конфигурации LiteBans интеграции
- Более понятные настройки подключения к БД LiteBans
- Chat ID в Telegram настройках теперь опциональный

#### База данных
- Улучшена поддержка SQLite для LiteBans интеграции
- Автоматический поиск SQLite БД в стандартных местах
- Улучшенная проверка существования таблиц LiteBans
- Поддержка разных типов БД (MySQL/SQLite) для LiteBans

#### Код
- Обновлена версия Java до 21
- Обновлена версия Paper API до 1.21
- Исправлены deprecated методы (URL конструктор)
- Улучшена обработка ошибок во всех компонентах
- Добавлена детальная информация в логи

### Исправлено

- Исправлена ошибка подключения к БД LiteBans при использовании той же БД
- Убраны требования к chat-id для работы Telegram бота
- Исправлено создание дубликатов сессий при неизменном статусе
- Улучшена проверка существования таблиц LiteBans для SQLite
- Исправлена обработка ошибок при подключении к БД

### Документация

- Обновлён README.md с информацией о новых возможностях
- Добавлена информация о Telegram командах
- Добавлена информация о LiteBans интеграции
- Обновлены требования к версиям Java и Minecraft

## [1.0.0] - 03-09-2025

### Первый релиз

#### Основной функционал
- Управление персоналом (добавление, удаление, просмотр информации)
- Система рангов и должностей
- Учёт рабочего времени с системой статусов
- Автоматическое управление статусами при входе/выходе
- Детальная статистика времени работы
- Поддержка различных типов хранения данных (YAML, MySQL, SQLite, H2)

#### Интеграции
- PlaceholderAPI - полная поддержка плейсхолдеров
- Telegram - базовые уведомления и привязка аккаунтов

#### Команды
- Управление персоналом: `/beaststaff add`, `/beaststaff remove`, `/beaststaff list`, `/beaststaff info`
- Учёт времени: `/beaststaff menu`, `/beaststaff status`, `/beaststaff stats`, `/beaststaff allstats`
- Административные: `/beaststaff setstatus`, `/beaststaff reload`
- Telegram привязка: `/beaststaff telegram bind`, `/beaststaff telegram unbind`

---

## Типы изменений

- `Добавлено` - для новых функций
- `Изменено` - для изменений в существующей функциональности
- `Устарело` - для функций, которые скоро будут удалены
- `Удалено` - для удалённых функций
- `Исправлено` - для исправления ошибок
- `Безопасность` - в случае уязвимостей
