# BeastStaff

Плагин для управления персоналом и учёта рабочего времени на Minecraft серверах.

## Поддерживаемые версии

- **Minecraft:** 1.16.5 - 1.21.11+
- **Java:** 21+ (рекомендуется)
- **Серверы:** Paper, Spigot, Bukkit
- **API версия:** 1.21

## Описание и возможности

BeastStaff - это плагин для Minecraft серверов, предназначенный для управления персоналом и учёта рабочего времени сотрудников.

### Управление персоналом
- Добавление и удаление сотрудников из системы
- Управление рангами и должностями персонала
- Автоматические уведомления о входе/выходе сотрудников
- Просмотр детальной информации о каждом сотруднике
- Гибкая система прав доступа

### Учёт рабочего времени
- Система статусов работы (В работе, AFK, Не в работе, Перерыв, Совещание, Обучение)
- Автоматическое управление статусами при входе/выходе с сервера
- Детальная статистика времени работы (день/неделя/месяц/всё время)
- Отслеживание активных рабочих сессий
- Административные инструменты для управления статусами

### Хранение данных
- **YAML файлы** - для небольших серверов
- **MySQL/MariaDB** - для крупных серверов с высокой нагрузкой
- **SQLite** - оптимальный выбор для средних серверов
- **H2** - встроенная база данных

### Интеграции
- **PlaceholderAPI** - полная поддержка плейсхолдеров для других плагинов
- **Telegram** - уведомления, персональные привязки аккаунтов и команды бота
  - Команды в Telegram: `/bs_ban_stats`, `/bs_recent_bans`, `/bs_status`, `/bs_set_status`
  - Работает без chat-id для личных сообщений
  - Поддержка отправки уведомлений в каналы/группы
- **LiteBans** - интеграция с системой наказаний
  - Статистика наказаний (баны, муты, кики, предупреждения)
  - Поддержка MySQL и SQLite баз данных
  - Кеширование статистики для оптимизации
- Настраиваемые сообщения и локализация

## Команды и права

### Основные команды

| Команда | Описание | Право доступа |
|---------|----------|---------------|
| `/beaststaff add <игрок> <ранг>` | Добавить сотрудника | `beaststaff.admin` |
| `/beaststaff remove <игрок>` | Удалить сотрудника | `beaststaff.admin` |
| `/beaststaff list` | Список всех сотрудников | `beaststaff.staff` |
| `/beaststaff info <игрок>` | Информация о сотруднике | `beaststaff.staff` |
| `/beaststaff reload` | Перезагрузить конфигурацию | `beaststaff.admin` |

### Команды учёта времени

| Команда | Описание | Право доступа |
|---------|----------|---------------|
| `/beaststaff menu` | Открыть меню статусов | `beaststaff.staff` |
| `/beaststaff status <статус>` | Изменить свой статус | `beaststaff.staff` |
| `/beaststaff stats [игрок] [дни]` | Просмотр статистики | `beaststaff.time.stats` |
| `/beaststaff allstats [дни]` | Общая статистика персонала | `beaststaff.time.admin` |
| `/beaststaff setstatus <игрок> <статус>` | Установить статус игрока | `beaststaff.time.admin` |

### Telegram команды (в игре)

| Команда | Описание | Право доступа |
|---------|----------|---------------|
| `/beaststaff telegram bind <ID>` | Привязать Telegram аккаунт | `beaststaff.staff` |
| `/beaststaff telegram unbind` | Отвязать Telegram аккаунт | `beaststaff.staff` |
| `/beaststaff telegram status` | Статус привязки | `beaststaff.staff` |

### Telegram команды (в боте)

| Команда | Описание | Требования |
|---------|----------|------------|
| `/bs_ban_stats` | Статистика наказаний за период | Требует LiteBans |
| `/bs_recent_bans` | Последние наказания | Требует LiteBans |
| `/bs_status` | Показать текущий статус | Требует привязку аккаунта |
| `/bs_set_status <статус>` | Изменить статус работы | Требует привязку аккаунта |

### Сокращённые команды
Все команды можно использовать с алиасом `/bs` вместо `/beaststaff`

### Права доступа

| Право | Описание | По умолчанию |
|-------|----------|--------------|
| `beaststaff.admin` | Полный доступ к командам плагина | op |
| `beaststaff.staff` | Доступ к командам персонала | op |
| `beaststaff.notify` | Получение уведомлений о входе/выходе | op |
| `beaststaff.time.admin` | Административные права учёта времени | op |
| `beaststaff.time.stats` | Просмотр статистики времени работы | op |

## Плейсхолдеры

Для использования с PlaceholderAPI доступны следующие плейсхолдеры:

### Статус и состояние
- `%beaststaff_status%` - текущий статус игрока
- `%beaststaff_status_color%` - цветовой код статуса
- `%beaststaff_is_working%` - работает ли игрок сейчас (Да/Нет)

### Статистика времени
- `%beaststaff_work_time_today%` - время работы за сегодня
- `%beaststaff_work_time_week%` - время работы за неделю
- `%beaststaff_work_time_month%` - время работы за месяц
- `%beaststaff_total_work_time%` - общее время работы
- `%beaststaff_current_session_time%` - время текущей рабочей сессии

### Цветовые коды статусов
- **В работе:** `§a` (зелёный)
- **AFK:** `§e` (жёлтый)
- **Не в работе:** `§c` (красный)
- **Остальные:** `§7` (серый)

## Установка на сервер

1. **Скачайте** последнюю версию плагина из [Releases](https://github.com/BeastMark441/BeastStaff/releases)
2. **Поместите** JAR файл в папку `plugins/` вашего сервера
3. **Перезапустите** сервер для создания конфигурационных файлов
4. **Настройте** конфигурацию в файле `plugins/BeastStaff/config.yml`
5. **Перезагрузите** плагин командой `/beaststaff reload` *(если не помогло, перезапустите сервер)*

### Дополнительные зависимости (опционально)
- **PlaceholderAPI** - для использования плейсхолдеров в других плагинах
- **MySQL/MariaDB** - для использования внешней базы данных
- **LiteBans** - для интеграции со статистикой наказаний
